# Proyecto-1
En este proyecto analicé datos históricos de ventas de videojuegos de la tienda online Ice, que comercializa títulos a nivel global. El dataset incluía información sobre reseñas de usuarios y críticos, géneros, plataformas (Xbox, PlayStation, entre otras) y clasificaciones por edad.

El objetivo fue identificar patrones y variables clave que influyen en el éxito comercial de un videojuego, con el propósito de detectar proyectos prometedores y generar insights útiles para la planeación de campañas publicitarias.

El análisis se realizó utilizando datos disponibles hasta el año 2016, simulando un escenario real en el que se debía planificar una estrategia de marketing para 2017. El enfoque principal del proyecto fue desarrollar experiencia práctica en análisis de datos, aplicando técnicas de limpieza, exploración y análisis estadístico que pueden extrapolarse a otros periodos.

El dataset contenía la columna rating, correspondiente a la clasificación ESRB (Entertainment Software Rating Board), la cual evalúa el contenido de los videojuegos y asigna categorías de edad como Teen o Mature. Esta variable fue utilizada para analizar su relación con las ventas y el desempeño comercial de los juegos.

## 🛠 Tools & Technologies
- Python  
- Pandas  
- NumPy  
- SciPy  
- Matplotlib / Seaborn  

## 🔍 Key Tasks Performed
- Data cleaning and preprocessing  
- Exploratory Data Analysis (EDA)  
- Sales trend analysis by platform and genre  
- Evaluation of user and critic reviews  
- Analysis of ESRB ratings and their impact on sales  

## 🎯 Results & Insights
Through this analysis, I identified trends related to platform popularity, genre performance, and the influence of reviews and age ratings on video game sales. These insights can be used to make data-driven decisions for future marketing and product strategies.
